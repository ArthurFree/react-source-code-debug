理解删除节点的操作之前，需要先认识删除节点的场景。

* 删除单个节点
```
                    Fiber树   

                   div#root 
                      |    
                    <App/> 
                      |  
 Delation   -->      div 

```

* 删除带有子树的节点，子树中的所有节点都要被删除。
```
                    Fiber树 

                   div#root 
                      |     
                    <App/>
                      | 
                     div
                      |
                   <Parent/>
                      |
 Delation   -->      span
                      |
                      P —————— <Child>
                                  |
                                  a
```

通过第二点我们不难想到，节点的删除注定需要遍历。现在直接以它为例，梳理一下删除过程。

因为删除需要该节点的父级DOM节点的参与，所以删除之前首先要定位到父级节点。过程就是以当前节点的
父节点为起点，找到第一个原生DOM节点。
